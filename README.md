# PDESoft26: BEM in NGSolve

View online: https://erdieee.github.io/pdesoft26/

## Abstract

Boundary element methods reformulate partial differential equations as integral equations on the boundary, reducing the dimension of the computational domain while producing dense system matrices. This presentation introduces boundary element methods in NGSolve and connects layer potential formulations with their implementation in the Python API. Particular attention is given to accurate integration of singular and nearly singular kernels, fast multipole acceleration, and the interface to HTool for hierarchical matrix compression. Numerical examples for Laplace, Helmholtz, and Maxwell problems demonstrate the available operators and workflows.
